Scoring Methodology
CRL Recovery Score (0–100)

Deficiency Type (30 pts) — CMC-only = 30, CMC + minor clinical = 20, Clinical efficacy = 10, Clinical safety = 5
Cash Runway (25 pts) — 24+ months = 25, 18-24 = 20, 12-18 = 15, 6-12 = 8, <6 = 0
Time Since CRL (15 pts) — <6 months = 15 (active remediation), 6-12 = 12, 12-18 = 8, 18-24 = 4, 24+ = 0 (stale)
Resubmission Filed (15 pts) — Yes + accepted = 15, Filed awaiting = 10, Not yet = 0
FDA Interaction (10 pts) — Type A meeting held = 10, Requested = 5, None disclosed = 0
Dilution Risk Offset (-5 pts) — Active ATM or shelf = -5, None = 0
483 Flag Penalty (-8 pts) — Form 483 at CMO within 18mo = -8
Repeat CRL Penalty (-10 pts) — CRL #2+ on same drug = -10

Historical resubmission approval rates: CMC-only ~85%, Manufacturing + labeling ~78%, Clinical efficacy ~45%, Clinical safety ~30%.
